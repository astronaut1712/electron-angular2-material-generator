# electron-angular2-material

Yeoman Generator for Electron Angular 2 with Material Design

## Install

`npm install -g yo generator-electron-angular2-material`

## How to use

```bash
$ mkdir your-new-app
$ cd your-new-app && yo
# start your app
$ npm start
# start with watch 
$ npm start-watch
```
